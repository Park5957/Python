파이썬이란?
파이썬은 1991년 네델란드의 수학자 '귀도 반 로섬' 에 의해 개발된 프로그래밍 언어로 '비단뱀'에서 유래한 로고를 볼 수 있다.

구글의 많은 앱들과 인스타그램 등 등의 신용적인 앱들도 파이썬으로 만등어졌다.

특징
1) 인터프리터 언어이다.
   컴파일러 언어는 (자바)

인터프린터 언어란?
코드가 한 줄씩 실행되는 언어 
//line by line 한 줄씩 실행 되어 결과를 바로바로 확인 할 수 있다.

컴파일러란?
코드를 모두 적은 뒤 실행 할 수 있는 언어

2) 플랫폼에 독립적이다.
   어떠한 운영체제에서도 같은 파이썬 코드를 실행할 수 있다. 다양한 운영체제 windows, osx, linux... 등등에서 동일한 코드로 동일한 경과를 볼 수 있다.
3) 객체 지향언어이다.
   Object criented 클래스와 객체를 기반으로 하는 언어로 정보은닉, 상속 등의 개념을 지원한다.

4) 동적 타이핑 dynamic typing 언어이다.
   프로그램 실행 중에 다른 작업을 처리할 수 있다.
5) 개발 속도가 빠르다.
   이미 완성된 많은 다양한 라이브러리, 데이터 분석 pandas, 수치연산 numpy, 시각화 matplotlib, Django, flask 웹개발, science kit learning 딥러닝 등등 을 사용하여 빠르게 프로그램을 작성할 수 있다.

*단점
하드웨어, 앱개발(아이폰, 안드로이드) 등에는 적용하기 어렵다. 

##############
# 주석 comment
##############
주석은 프로그램 소스 코드에 설명을 작성한 것을 의미한다. 주석은 프로그램 실행에 전혀 영향을 끼치지 않는다. 유지 보수나 팀 작업을 할 때 모두가 이해하기 쉽고 의미 전달이 잘 되는 주석을 작성하는 것이 바람직하다.
파이썬에서 주석은 #으로 시작하거나 ‘’’  ‘’’ 삼중 따옴표로 표현한다.

##############
# 변수 Variable
##############
변수는 어떤 데이터를 저장할 때 사용하는 메모리 저장소(?) 이다.
(온전한 이유는 아니지만 전체적인 의미는 맞음)
변수를 만들 때는 = 등호 왼쪽에 변수명을 작성하고 등호 오른쪽에는 저장할 값을 작성한다.
변수명 규칙
1.   영문, 한글, 숫자, 밑줄로 구성된다.(단, 한글은 변수명으로 사용하지 말 것)
2.   특수문자는 사용할 수 없다.(! @ # $ % 등등) (단, _ 는 사용이 가능함.)
3.   대문자와 소문자를 구분한다.
4.   변수명의 첫 글자는 숫자를 사용할 수 없다.
5.   예약어 (print, list, dict, if, for 등등) 은  변수명으로 사용할 수 없다.
*권장사항(필수는 아님)
가급적 변수명은 소문자만 사용한다.
변수명으로 한글은 사용하지 말자.
의미 있는 변수명을 사용하자.

##############
# 슬라이싱 indexing
##############
문자열, 리스트, 튜플 모두 동일
한 문장 이상으로 구성된 단어나 문장들을 추출할 때 사용한다.

예시
print(변수명[값:값2])
값2는 해당 문자열 미만까지 표기를 해준다.
